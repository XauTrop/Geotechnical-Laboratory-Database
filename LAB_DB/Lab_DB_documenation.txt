##########  Database stored information  ##########
sample_id: Sample name (text)
sample_depth: Sample depth mbsf (float)
core_id: Core name (tetx)
core_depth: Core depth mbsl (float)
lat: Core Latitude (decimal degrees float)
lon: Core Longitude (decimal degrees float)
cruise: Cruise name (text)
test_type: Test type RB, CRS, TAS (text)
grain_size: Grain size (CL, ST, SD) (text)
pl: Plastic index (float)
ll: Liquid index (float)
pi: Plasticity index (float caluculated)
is_wc: In situ water content (float)
is_poro: In situ porosity (float)
is_effstrs: In situ effective stress (float)
dep_poro: porosity on deposition (float)
depo_hydcon: Hydraulic conductivity on deposition (float)
depo_store: Specific storage on deposition (float)
cc: Compressibility index (float)
cs: Undoading index (float)
cr: Recompression index (float)
strain_rate: Strain rate for CRS (float)
pre_strs: Pre-consolidation effective stress (float)
